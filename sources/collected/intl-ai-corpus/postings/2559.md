The Opportunity

As an IT Automation and AI Engineer at Okta, you will be a
key contributor to the team building the platforms and
services that drive our internal AI transformation. We are
moving beyond simple automation to build sophisticated,
"agentic" workflows—utilizing LangGraph and Gemini
Enterprise to create stateful, reasoning-capable systems
that support our global workforce.

This is a hands-on role for a professional engineer who
excels at writing production-grade Python and is eager to
bridge the gap between AI research and enterprise-scale
applications. You will own the implementation of core
features, orchestrating complex LLM interactions while
ensuring performance, security, and reliability.

What You’ll Be Doing

- Build Agentic Workflows: Develop and maintain stateful
  multi-agent systems using LangGraph to solve complex, non-
  linear business tasks.
- Gemini Integration: Implement and optimize LLM interactions
  using Gemini Enterprise, focusing on prompt engineering,
  function calling, and structured outputs.
- State & Memory Management: Design and manage agent state,
  persistence, and "human-in-the-loop" checkpoints to ensure
  reliable automated decision-making.
- Backend Development: Write high-quality, production-ready
  Python code to build the APIs and services that power our
  internal AI ecosystem.
- RAG & Grounding: Work on Retrieval-Augmented Generation
  (RAG) pipelines to ensure agents have access to accurate,
  real-time enterprise context.
- Infrastructure & Deployment: Deploy and manage services on
  GCP using containerized environments (Docker/Kubernetes)
  and Infrastructure as Code (Terraform).
- Operational Excellence: Promote engineering best practices
  around testing AI logic, security, and CI/CD to ensure
  services are production-ready.

What You’ll Bring to the Role

- Professional Python Core: 2–4 years of software engineering
  experience with a deep command of Python and its
  asynchronous ecosystem.
- AI Framework Mastery: Hands-on experience with LangGraph (or
  similar frameworks like LangChain). You understand how to
  build cycles, nodes, and conditional edges in a stateful
  application.
- LLM Practicality: Experience working with frontier models
  like Gemini or GPT-4. You know how to manage prompt
  versioning, token constraints, and model behavior.
- System Design Foundations: Solid understanding of
  distributed systems, cloud-native architectures, and
  building/consuming RESTful APIs.
- Problem-Solving Autonomy: The ability to take a defined
  workflow or prototype and drive it to production with
  minimal oversight, knowing when to escalate complex trade-
  offs.
- Collaborative Spirit: Strong communication skills and a
  desire to work across team boundaries to deliver cohesive,
  high-impact AI solutions.

Why Join Okta

- Build With Purpose: Work on meaningful problems that
  simplify and secure how Okta operates at a global scale.
- Cutting-Edge Stack: Work with the latest from Google’s AI
  suite (Gemini) and the leading edge of agentic
  orchestration (LangGraph).
- Grow in a Technical Culture: Join a supportive, high-caliber
  team that values technical excellence, clean code, and
  personal development.
