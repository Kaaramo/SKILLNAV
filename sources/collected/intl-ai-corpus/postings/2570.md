## Description complète

The Software Engineer (AI Backend Services) will design,
develop, and own backend systems and components that power
generative AI applications and large language model (LLM)
integrations. This role focuses on independently
architecting and delivering scalable, secure, and high-
performance services using Python, FastAPI, and AWS cloud
infrastructure, while mentoring junior engineers. This is an
individual contributor role KEY RESPONSIBILITIES

- Design and implement Python-based backend services (on
  FastAPI) for AI workloads, including LLM integration and
  generative AI pipelines
- Own end-to-end design and delivery of APIs and microservices
  for model serving, inference, and orchestration of AI
  agents
- Architect and maintain AWS-based infrastructure for AI
  workloads (e.g., Lambda, EKS) with focus on scalability
  and reliability
- Make independent technical decisions for component-level
  architecture and design patterns
- Lead integration efforts for LLMs and generative AI models
  into production systems using LangChain, Langfuse, and
  litellm
- Establish monitoring, logging, and observability best
  practices for AI backend services
- Optimize backend performance for large-scale AI applications
- Mentor Software Engineer II team members through code
  reviews and technical guidance
- Collaborate with cross-functional teams and influence
  technical direction across multiple projects

- 4-7 years of professional software engineering experience

- Deep knowledge of microservices architecture, API design,
