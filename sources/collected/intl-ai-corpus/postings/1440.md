## Description complète

Job Title: Azure AI / ML Engineer

Our organization is committed to advancing enterprise
capabilities by leveraging Azure AI and Machine Learning to
solve complex business challenges and accelerate product
delivery. We are seeking a Senior AI / ML Platform Engineer
to design and implement enterprise‑grade data and AI/ML
platform services across Azure and GCP.

The ideal candidate brings hands-on experience delivering
AI/ML cloud solutions within regulated industries—preferably
financial services—with a strong understanding of security,
compliance, and operational best practices. This role
focuses on AI/ML service enablement, automation, and
applying AIOps principles to build a scalable, secure,
self‑service platform.

You will work collaboratively with cloud engineering,
network, security, and risk teams to deliver secure Azure AI
and ML solutions aligned with enterprise security and
compliance standards. You will also partner closely with
developers and business units to continuously improve
frameworks, tooling, and automation for AI/ML services.

Roles and Responsibilities

- Demonstrate strong passion for Generative AI and Machine
  Learning
- Apply an AIOps/MLOps mindset to enable services and build
  pipelines for training and production models
- Partner with application development teams to leverage AI/ML
  services that solve business objectives
- Ensure secure handling of data, model training, and
  deployment within cloud environments
- Develop APIs and Webhooks using Python, Git Actions, and
  Terraform
- Write and maintain Azure Resource Manager (ARM) and
  Terraform templates
- Implement and support Azure security features (data
  protection, authentication, RBAC, etc.)
- Manage Public Key Infrastructure (PKI), including
  certificate handling for Azure PaaS services
- Troubleshoot Azure services, DNS, connectivity, NSGs, and
  routing
- Automate cloud operations using native Azure CLI
- Support platform and analytics deployments via CI/CD
  pipelines
- Ensure all cloud solutions adhere to internal security and
  compliance policies
- Develop and consume APIs, SDKs, and Webhooks for integration
  across enterprise systems, DevOps tools, and cloud
  platforms
- Operate effectively within fast‑paced DevOps teams following
  Scrum methodologies
- Demonstrate strong critical thinking, research, and
  analytical abilities
- Work independently and collaboratively with a positive,
  proactive attitude

Required Skills

- 3+ years as an AI and ML Platform Engineer
- 3+ years developing platform orchestration code using Azure
  Python SDK, Terraform, and GitHub Runners
- Strong experience delivering Cloud Infrastructure as Code
  (IaC) via CI/CD pipelines, Terraform, and Git Actions
- Knowledge of GenAI models, development frameworks (e.g.,
  LangChain, Semantic Kernel), and AI ecosystem tools
  (evaluation, monitoring, governance, security, vector
  databases)
- Ability to identify and mitigate security risks associated
  with Generative AI
- Understanding of RAG architecture components, usage
  patterns, and optimization
- Demonstrated self‑starter with the ability to take
  initiative and drive results
- Advanced critical thinking, analytical, and problem‑solving
  skills
