Job Title: Senior Software Engineer — Full Stack (AI-Powered

We are looking for a talented and passionate Senior Software

Full Stack Development:

- Design, develop, and maintain scalable, high-performance web

- Build and consume RESTful APIs and GraphQL services
- Develop reusable, modular, and maintainable code following

- Collaborate with UI/UX designers to translate wireframes

- Optimize application performance, scalability, and security

AI-Powered Development:

- Leverage AI coding tools (GitHub Copilot, ChatGPT, Cursor,

- Integrate AI/ML APIs and LLM-based features (e.g., OpenAI,

- Build intelligent features such as smart search,
  recommendations, chatbots, or automation workflows
- Stay updated with the latest AI coding trends and evaluate
  new tools for team adoption

Architecture & Leadership:

- Lead technical design discussions and contribute to system
  architecture decisions
- Conduct thorough code reviews and mentor junior developers
- Drive best practices around coding standards, CI/CD, and
  DevOps
- Collaborate closely with Product Managers, QA, and cross-
  functional teams
- Break down complex requirements into actionable technical

Must Have:

- 5–10 years of hands-on experience in full stack development
- Strong proficiency in React.js — hooks, context API, Redux,
  Next.js
- Strong proficiency in Node.js — Express.js, REST APIs,
  microservices
- Experience with databases — MongoDB, PostgreSQL, or MySQL
- Proficiency in JavaScript (ES6+) / TypeScript
- Experience with Git, GitHub/GitLab, branching strategies
- Familiarity with AI coding assistants (GitHub Copilot,
  Cursor, etc.)
- Experience integrating third-party APIs and AI/LLM APIs
- Understanding of cloud platforms — AWS / Azure / GCP

Good to Have:

- Experience with Docker, Kubernetes, or CI/CD pipelines
- Knowledge of WebSockets, Redis, or message queues
