## Description complète

About the Role

Join Planera to build AI-powered agentic workflows that
transform construction planning and scheduling. You'll
design intelligent automation systems, conversational
interfaces, and autonomous agents that help construction
teams work more efficiently. This is a product engineering
role focused on developing practical AI solutions.

Key Responsibilities

- Design and implement agentic AI workflows for construction
  automation
- Build intelligent agents for scheduling, resource
  allocation, and project optimization
- Develop conversational interfaces and prompt engineering
  solutions
- Create robust backend services in Python/Golang to support
  AI features
- Integrate LLMs and AI APIs into construction workflows
- Build scalable APIs for complex construction project data
- Collaborate with frontend team to deliver seamless AI
  experiences

Requirements

- 4+ years full-stack development (backend-focused)
- Strong proficiency in Python and/or Golang (Node.js is a
  plus)
- Experience with LLMs, prompt engineering, and AI APIs
- Understanding of agentic systems and workflow automation
- Detailed knowledge of agent frameworks like Langchain
- Experience building conversational AI or chatbots
- Have deployed agentic chat systems to production
- Strong API design and database skills (SQL/NoSQL)
- Cloud platform experience (AWS/GCP/Azure)
- React knowledge for frontend integration
- Product mindset with focus on user outcomes
- Excellent remote communication skills

Preferred

- Construction tech or project management software experience
- Knowledge of RAG systems and vector databases
- Experience with workflow orchestration tools
- Understanding of construction industry workflows

Tech Stack

Python, Golang, React, PostgreSQL, AWS/GCP, Docker, GitLab,
Datadog
