### Raisonnement classification (LLM upstream)

This role is a cloud/DevOps infrastructure engineer that
supports AI workloads rather than building or training
models directly. The engineer manages Azure infrastructure
(IaC, automation, containers) that hosts AI services like
Azure OpenAI and Azure ML, but does not fine-tune models,
build RAG pipelines, or develop AI agents. The core work is
platform operations and infrastructure automation enabling
AI services to run in production.

About the Role

We are looking for a Cloud Engineer to join our team and
operate cloud infrastructure with a strong focus on
Microsoft Azure and AI/ML-related solutions. This role

platform (IaC) that supports AI services and workloads—from
prototyping to production.

What You’ll Do

- Working with the Microsoft Azure cloud platform, including
  designing, deploying, and optimizing infrastructure.
- Contribute to the design and operation of AI-related
  components in Azure (depending on the team’s scope), such
  as:Azure OpenAI, Azure AI Search, Azure Machine Learning,
  AKS for inference/serving, and event-driven integrations.
- Leverage expertise in AWS and, ideally, GCP to ensure
  comprehensive cloud platform management.
- Implement Infrastructure as Code (IaC) methodologies.
