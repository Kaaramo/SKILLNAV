## Description complète

We are an internal Cyber Red Team pioneering the use of
Agentic AI to simulate real-world adversaries. This AI-
driven role will heavily be dependent on automating real-
world attack workflows using Python and LLMs. We are looking
for a strong Python engineer who is curious about security
and wants to work on automating offensive techniques using
AI. Cyber Security background is NOT mandatory. You will
work directly with experienced Red Team professionals to
convert real attack logic into scalable, intelligent
systems.


Responsibilities

RESPONSIBILITIES:

- Build Python-based automation around offensive security
  tools
- Integrate LLM APIs (OpenAI/Anthropic) into real workflows

- Hands-on experience with LLM APIs
