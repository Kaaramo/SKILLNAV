Position Overview

Autodesk is seeking a highly motivated, experienced
Principal Engineer to join high‑impact teams building the
next generation of AI‑powered Test Automation solutions for
Cloud and Desktop application.

We are looking for a strong technical contributor with
expertise in automation, cloud services, and AI agent
development. As an ideal candidate, you have built scalable
applications and test automation frameworks in fast-paced,
agile environments. You are comfortable working across
cloud, desktop, and web technologies, and bring hands-on
experience integrating AI-driven solutions into modern
workflows. This is an exciting opportunity to contribute to
the evolution of engineering excellence in a lean/agile
environment and help deliver high-quality, innovative
solutions.

Responsibilities

- Contribute to software design and architecture across
  desktop and cloud platforms
- Develop and maintain end-to-end and integration Agentic test
  automation using Playwright, Cypress, Python, and
  JavaScript
- Integrate AI agents and LLMs into development and QA
  workflows (RAG, MCP, FastAPI)
- Leverage AWS services to design scalable environments
- Embed automated tests into CI/CD pipelines using Jenkins
- Collaborate with stakeholders to define priorities,
  roadmaps, and delivery plans
- Ensure engineering efficiency by identifying opportunities
  and driving improvements
- Work in Agile teams, using JIRA and XRAY for tracking and
  reporting

Minimum Qualifications

- BS degree in Computer Science, Software Engineering, or
  related field
- 9-14 years of professional experience in software
  development
- Strong programming skills in Python, JavaScript, TypeScript
- Hands-on experience with Playwright, Cypress, Pytest
- Experience with AWS cloud services for scalable
  environments.
- Knowledge of CI/CD pipelines, Jenkins, Git
- Familiarity with Agile methodologies and defect tracking
  tools
- Strong communication and collaboration skills

AI Technical Skills

- Experience building and deploying AI agents using LangGraph
  or similar frameworks
- Hands-on with LLM integration, prompt engineering, and RAG
  patterns
- Ability to design AI-driven test automation systems for
  UI/API testing
- Strong Python programming with async/await patterns and
  scalable API design
- Knowledge of MCP or similar agent communication protocols
