## Description complète

The Senior Engineer is a hands-on builder who operates as an
AI-augmented developer. You will take ownership of discrete
workstreams within larger AI-enabled product initiatives and
leverage agentic coding systems to accelerate design,
development, and testing.



This is not a narrow frontend or backend role. You are
expected to operate across the stack and across the
development lifecycle.



Core Responsibilities:



1. Full Workstream Ownership

- Translate product requirements into technical design and
  implementation
- Build both frontend and backend components
- Integrate AI services and model APIs into applications
- Ensure secure and compliant handling of data



2. Agentic Build Execution

- Use AI coding assistants to:
  - Generate initial code scaffolds
  - Refactor and modularize components
  - Produce automated test cases
  - Accelerate documentation
- Coordinate multiple agents to parallelize development tasks
  where appropriate
- Apply disciplined review and validation processes to all AI-
  generated outputs



3. Rapid UX & Testing Acceleration

- Create rapid prototypes and iterate on UI/UX flows using AI-
  assisted design tools
- Generate and execute automated test coverage
- Support performance testing and debugging using AI
  diagnostics



4. Workflow-Centric Development

- Work directly with advisory professionals to understand
  workflow bottlenecks
- Build solutions that integrate seamlessly into daily
  practitioner workflows
- Optimize for usability, speed, and minimal cognitive
  friction



Qualifications:



Required Experience

- 3–7 years of full-stack development experience
- Experience shipping production-ready applications
- Strong understanding of APIs, databases, and frontend
  frameworks



AI Fluency

- Active user of advanced coding assistants and agentic
  development tools
- Experience leveraging AI for testing, debugging, or code
  generation
- Understanding of prompt engineering for development
  workflows



Preferred

- Experience in enterprise or regulated environments
- Familiarity with integrating LLM APIs into applications
- Comfort working in ambiguous, fast-moving build environments
