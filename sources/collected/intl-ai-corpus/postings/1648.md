## Description complète

Experience: 8+ yrs for Junior level / 12 + years for Senior
level

Role Overview

We are hiring an experienced L2 Application Support Engineer
to support a cloud-based Enterprise AI/GenAI platform built
on .NET Core and Microsoft Azure. The role involves
production support, troubleshooting API & authentication
issues, and ensuring platform stability across environments.

Key Responsibilities

- Handle L2 production incidents and service requests
- Troubleshoot .NET Core / REST API-based applications
- Support AI/LLM integrations and model endpoint connectivity
- Diagnose API failures, 403/authentication issues, and
  timeout/latency problems
- Manage API keys, IP whitelisting, and access provisioning
- Monitor applications using Azure Application Insights / Log
  Analytics
- Support Dev/QA/Prod environments and release validation
- Coordinate with L3, AI, and DevOps teams for escalations

Required Skills

- Strong experience in .NET Core / ASP.NET Core application
  support
- Experience supporting REST APIs and microservices
- Azure App Services / Container Apps / AKS (basic exposure)
- Azure AD / SSO troubleshooting
- Log analysis & production incident handling
- SQL for troubleshooting

Preferred

- Exposure to AI/LLM-based platforms
- Understanding of token-based API usage models
- Azure certification (AZ-900 / AZ-104)
- ITIL knowledge
