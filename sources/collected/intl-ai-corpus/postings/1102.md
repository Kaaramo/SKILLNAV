Role: Senior UI Developer – Frontend Platform

We are seeking a Senior UI Developer to join Arctic Wolf’s
R&D organization, where you will play a critical role in
shaping the user experience of our security operations
platform.

In this role, you will bridge UX design and agile
engineering to deliver intuitive, performant, and accessible
user interfaces used by security practitioners around the
world. You will help define frontend architecture,
contribute to our design system, and bring deep frontend
expertise to help teams make informed, scalable design and
technical decisions.

This is a hands-on role for a frontend engineer who is
passionate about user experience, modern web technologies,
and building high-quality interfaces at scale.

About the Role

- Design, build, test, and maintain modern web interfaces
  using React and TypeScript.
- Translate UX designs and interaction models into production-
  ready, reusable UI components.
- Implement frontend architecture, state management, and
  performance optimizations for large-scale applications.
- Build and maintain shared utilities, hooks, and component
  libraries to enable code reuse across teams.
- Contribute to and evolve the Arctic Wolf Design System,
  ensuring consistency, scalability, and usability.
- Champion accessibility best practices, delivering inclusive
  interfaces that work across devices and input methods.
- Collaborate closely with UX designers, product managers, and
  backend engineers throughout the full software development
  lifecycle.
- Apply test-driven development practices and ensure code
  quality through automated testing.
- Participate in code reviews, architectural discussions, and
  continuous improvement initiatives.
- Mentor junior developers and contribute to team growth
  through technical leadership and knowledge sharing.

About You

- 5+ years of experience building modern frontend applications
  in production environments.
- Strong expertise with React, TypeScript, and modern
  JavaScript (ES modules).
- Experience with Web Components, Service Workers, and
  Progressive Web Applications (PWAs).
- Deep understanding of HTML5, CSS, and CSS preprocessors such
  as Sass/Less/Stylus.
- Solid knowledge of frontend performance optimization and
  cross-browser compatibility.
- Experience with accessibility standards (WCAG) and inclusive
  design practices.
- Working knowledge of GitHub, CI/CD pipelines (Jenkins
  preferred), and automated testing tools.
- Strong testing mindset, including unit, integration, and UI
  testing.
- Ability to lead frontend initiatives and influence
  architectural decisions.
- Passion for building clean, intuitive, and maintainable user
  experiences.
