About Accellor

Accellor is a trusted AI and digital engineering partner
helping enterprises build scalable, outcome-driven solutions
using modern cloud, data, and AI technologies. We work
across Financial Services, Retail, High Tech, and
Healthcare, combining deep domain expertise with strong
engineering and applied AI capabilities to deliver
measurable business impact.

Role Overview

We are looking for a Software Engineer who has atleast 6
months experience building production-grade Agentic AI and
multi-agent systems. This role is ideal for someone who can
design complex systems end-to-end—spanning frontend,
backend, AI orchestration, and cloud infrastructure—and
mentor teams in building AI-native applications.

Key Responsibilities

- Design and build full-stack applications using modern
  frontend and backend frameworks.
- Design and develop multi-agent AI systems using frameworks
  such as LangChain, LangGraph, AutoGen, MAF, or similar.
- Build AI-native applications combining agentic workflows,
  traditional ML, and rules-based systems.
- Develop scalable frontend applications using React, Next.js,
  and micro-frontend SPA architectures.
- Build backend services using Node.js, Java (Spring Boot /
  Kotlin), GraphQL, tRPC, and REST APIs.
- Design Backend-for-Frontend (BFF) layers optimized for AI-
  powered user experiences.
- Integrations between frontend, backend, AI services, and
  enterprise systems.
- Drive best practices for code quality, performance,
  security, and scalability.
- Mentor engineers and contribute to technical roadmap
  planning.

Requirements

- 1- 3 Years of experience as a Full Stack Engineer.
- Minimum 6 months building Agentic AI or multi-agent
  applications.
- Good frontend knowledge with skills like React, Next.js,
  JavaScript/TypeScript.
- Good backend knowledge with skills like Node.js and Java
  (Spring Boot / Kotlin).
- Hands-on experience with agent frameworks and orchestration
  patterns.
- Knowledge with BFF design patterns and API gateway
  configurations.
- Cloud experience (AWS, Azure, or GCP).
- CI/CD, Docker, and Kubernetes experience.
