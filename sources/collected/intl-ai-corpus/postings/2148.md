## Description complète

Job Description:

We are looking for a heavy-hitting Python Developer with
deep Azure Infrastructure expertise to own the backend and
deployment for a production-grade AI conversational
platform. This isn't a research role; it’s a "boots-on-the-
ground" engineering position requiring you to architect,
deploy, and integrate AI services into a complex enterprise
ecosystem for a Fortune 500 connectivity leader.



The Mission

You will join an ongoing program for a massive global
telecommunications and connectivity client. The platform—a
sophisticated chat interface with streaming responses and
long-term memory—has been architected and built solo by our
Technical Director to date. As we approach an April 2025
production launch, you will take over the day-to-day
engineering, architecture, and "Full-Stack DevOps" ownership
to scale the platform through the end of the year.

What You’ll Do (The Role)

- Architect from Scratch: Design and deploy cloud
  infrastructure within Azure, ensuring services are
  resilient, monitored, and secure (RBAC).
- Engineer Backend Services: Develop and extend high-
  concurrency APIs using Python and FastAPI to support
  agentic behaviors and real-time streaming.
- Own the Deployment: In the absence of a centralized CI/CD
  team, you will manage the full deployment lifecycle,
  running scripts across environments and building the path
  toward future automation.
- Integrate AI as a Service: Implement and orchestrate LLMs,
  Vector Databases, and RAG pipelines using the Microsoft
  Agent Framework (MAF).
- Data Orchestration: Build the data flows required for "agent
  memory" and prepare for upcoming integrations with Adobe
  Experience Platform (AEP) and Salesforce Data Cloud.
- Navigate the Enterprise: Work within a highly matrixed
  environment, independently managing communications with
  upwards of 25 project managers and stakeholders.

What You’ll Have (The Requirements)

- Python Mastery: Professional experience building production
  backend systems (FastAPI preferred).
- Azure Infrastructure: You are a "Cloud First" developer
  capable of configuring Azure Functions, Container Apps,
  and API Management independently.
- The "DevOps" Skillset: Hands-on experience with Docker and
  manual environment management; you don't need a DevOps
  team to ship code.
- Practical AI Understanding: You treat AI as "just another
  development effort." You understand how to integrate LLMs
  and manage context without getting lost in the "hype."
- Enterprise Endurance: Experience working in large-scale
  corporate environments.

The Tech Stack

- Languages/Frameworks: Python, FastAPI, Microsoft Agent
  Framework (MAF)
- Cloud: Azure (Functions, APIM, RBAC, Monitoring)
- Data/Containers: Postgres, Docker, Vector Databases
- Future Integrations: Adobe Experience Platform, Salesforce
  Data Cloud
