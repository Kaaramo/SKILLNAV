## Description complète

Job Summary Synechron is seeking a highly skilled Full Stack
Developer to design, develop, and optimize scalable, secure,
and intelligent applications. This role requires a strong
focus on backend API development, cloud deployment, and
AI/ML integration, supporting the organization’s digital
transformation and innovation initiatives. The successful
candidate will work across front-end and back-end domains,
collaborating with cross-functional teams to deliver
impactful solutions that enhance business operations and
support future growth.

Software Requirements

- Required:
  - Proficiency in frontend frameworks such as React, Angular,
    or Vue.js (latest stable versions)
  - Strong backend skills in Golang and Node.js (latest stable
    releases)
  - Experience designing and developing RESTful APIs,
    microservices, and serverless architectures
  - Deep knowledge of AWS services including EC2, Lambda, S3,
    API Gateway, and CloudFormation for deployment and
    management
  - Familiarity with containerization (Docker), orchestration
    (Kubernetes), and CI/CD pipelines
  - Exposure to AI/ML model integration tools and frameworks
    (TensorFlow, PyTorch, or similar)

- Preferred:
  - Experience with GraphQL or alternative API standards
  - Advanced AWS deployment techniques and security best
    practices
  - Knowledge of data engineering tools and pipelines

Overall Responsibilities

- Design, develop, and maintain scalable web applications,
  APIs, and microservices leveraging modern frameworks and
  cloud infrastructure
- Deploy, monitor, and optimize cloud-based applications for
  performance, reliability, and security
- Collaborate with product teams, data scientists, and
  infrastructure teams to build intelligent systems
  integrating AI and ML models
- Conduct system architecture planning, code reviews, testing,
  and documentation to ensure high standards of quality and
  maintainability
- Support and troubleshoot deployed applications, ensuring
  operational stability and continuous improvement
- Keep current with emerging industry trends and incorporate
  relevant innovations into solutions

Technical Skills (By Category)

- Programming Languages:Required: Golang, Node.js,
  JavaScript/TypeScript for front-end; Python for AI/ML
  integrationPreferred: Additional scripting languages such
  as Bash or PowerShell for automation

- Databases/Data Management:Experience with cloud-native data
  storage solutions, relational databases (PostgreSQL,
  MySQL), and knowledge of NoSQL options (e.g., DynamoDB,
  MongoDB)

- Cloud Technologies:AWS (EC2, Lambda, S3, API Gateway,
  CloudFormation), experience in deploying scalable cloud
  applications and serverless architecture

- Frameworks and Libraries:React, Angular, Vue.js, Express.js,
  AWS SDKs, TensorFlow, PyTorch

- Development Tools and Methodologies:Git, Jenkins, Docker,
  Kubernetes, Agile and Scrum methodologies, CI/CD pipelines
  using tools like Jenkins or AWS CodePipeline

- Security Protocols:Best practices in API security, data
  encryption, IAM policies, and secure cloud deployment

Experience Requirements
