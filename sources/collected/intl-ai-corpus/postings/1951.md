## Responsabilités

- Analyze aircraft fleet operational data to design and prototype diagnostic and prognostic solutions for aircraft sub-systems
- Apply Prognostics & Health Monitoring techniques to assess health state and monitor component failures for aircraft systems
- Develop algorithms to estimate Remaining Useful Life (RUL) and identify early-stage degradation patterns
- Implement anomaly detection, fault isolation and health prediction using physics, probabilistic, and machine learning approaches in Python and MATLAB/Simulink
- Build and productionize ML models (anomaly detection, RUL, fault classification) using TensorFlow, PyTorch, scikit-learn with MLOps practices
- Implement edge/cloud architectures and scalable storage for fleet-scale telemetry and real-time analytics
- Apply explainable AI, privacy-preserving methods, and federated learning where appropriate
- Execute AI testing, verification & validation including simulation, adversarial checks, and post-deployment monitoring
