## Responsabilités

- Build and maintain multi-agent workflows from solution design documents into production-ready implementations
- Implement tool integrations with APIs, ERP, CRM, ITSM with error handling and graceful degradation
- Build end-to-end RAG pipelines including ingestion, chunking, embedding, vector store and retrieval evaluation
- Monitor and improve retrieval quality and integrate RAG as knowledge backend for agents
- Develop and maintain RPA task bots and implement AI agent to RPA handoff logic
- Write unit and integration tests and contribute to CI/CD pipelines and prompt versioning
- Monitor deployed systems and participate in incident response for agent failure modes
- Participate in stakeholder use-case discovery and document pipelines and agent configurations
