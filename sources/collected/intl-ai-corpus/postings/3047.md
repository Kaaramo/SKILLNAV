## Description complète

We’re looking for a highly effective Software Engineer who
leverages modern AI tools to dramatically accelerate
development speed, code quality, and problem-solving.

This role is not about training models—it’s about using AI
(e.g., code generation, copilots, agents, and automation
tools) to build better software, faster. You should be
comfortable treating AI as a core part of your workflow.

What You’ll Do

- Use AI-powered development tools (e.g., copilots, code
  assistants, automated testing tools) to design, write, and
  refactor code

- Rapidly prototype and ship features with the help of AI-
  assisted workflows

- Translate product requirements into working software with
  high velocity

- Validate, debug, and improve AI-generated code to production
  standards

- Build internal tools and automations that leverage AI to
  improve team productivity

- Continuously evaluate and adopt new AI tools to enhance
  development workflows

- Collaborate with product and design teams to deliver high-
  quality features quickly

- Maintain strong code quality, testing, and documentation
  practices—even when moving fast

What You’ll Need

- Ability to be a self-starter, self-manager, and leader. You
  know the highest priorities and get them done without
  being told what to do.

- 5+ years of software engineering experience

- Strong proficiency in: .NET 8, C# 9, React, SQL Server

- Demonstrated experience using AI tools (e.g., GitHub
  Copilot, ChatGPT, code generators) in daily development

- Ability to critically evaluate and refine AI-generated
  outputs

- Familiarity with automation tools, scripting, or workflow
  orchestration

- Experience integrating AI APIs into applications
