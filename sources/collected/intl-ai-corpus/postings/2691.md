This role involves directly building a RAG (Retrieval-
Augmented Generation) pipeline that integrates with an LLM
to recommend database configuration changes. The intern will
be hands-on with GenAI techniques—specifically RAG
architecture and LLM integration—making this an ai-first
role.

## Responsabilités

- Design and build a RAG pipeline to retrieve system performance information and administrative documents
- Integrate retrieved data with an LLM to generate configuration change recommendations for HCL Informix Database Server
- Work with database performance metrics and documentation as knowledge sources for the RAG system
- Present capstone project outcomes in the Internship Showcase at the end of the 12-week program
