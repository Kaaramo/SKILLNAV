### Raisonnement classification (LLM upstream)

This role directly involves training large language models
through RLHF — the engineer compares, ranks, repairs, and
refactors AI-generated code and injects feedback into the
RLHF pipeline. While the engineer isn't building model
architectures, they are directly participating in the model
training loop by providing expert signals that tune LLM
behavior, making this an ai-first role.

What You'll Be Doing

Help train large language models (LLMs) to write production-
grade code across a wide range of programming languages:

- Compare and rank multiple code snippets, explaining which is
  best and why
- Repair and refactor AI-generated code for correctness,
  efficiency, and style
- Inject feedback (ratings, edits, test results) into the RLHF
  pipeline and keep it running smoothly

End result: the model learns to propose, critique, and
improve code the way you do.

RLHF in one line: Generate code → expert engineers rank,
edit, and justify → convert that feedback into reward
signals → reinforcement learning tunes the model toward code
you'd actually ship.
