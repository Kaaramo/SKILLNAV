## Responsabilités

- Implement and optimize sophisticated Agentic RAG and standard RAG pipelines with advanced prompt engineering
- Design, develop, and deploy autonomous multi-agent systems using frameworks like CrewAI to solve complex business problems
- Implement and refine Agent-to-Agent (A2A) communication protocols for collaborative task execution between multiple AI agents
- Build, test, and maintain MCP connectors to enable AI agents to interact with data sources like Microsoft Spreadsheets, Databricks Lakehouse, and Postgres databases
- Create robust and scalable REST APIs to serve as the backbone for AI bots and applications
- Enable deployment, management, and monitoring of AI applications within Open Telekom Cloud (OTC) cloud environment
