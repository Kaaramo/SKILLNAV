## Description complète

We need a self-driven, curious, and adaptable individual who
quickly picks up new tools and delivers outcomes. Strong SQL
proficiency is a must, along with an excellent understanding
of dimensional modeling, data integration, data securit/
governance, and a foundational understanding of Generative
AI. Below are the basic skill expectations:

- Very Strong SQL skills for data extraction, transformation,
  and metric creation.
- Working understanding of data modeling concepts (dimensions,
  facts, hierarchies).
- Knowledge of data validation and profiling principles.
- Should be able to work on system integrations with various
  external tools - AD, Jira, Service Now etc.
- Eagerness to learn new technologies quickly.
- Foundational understanding of Generative AI models, LLMs,
  MCP and prompt engineering.
- Exposure to APIs, basic Python or scripting for automation.
- Good communication and analytical thinking skills.
- Collaborative mindset focused on helping the broader team
  succeed.
- Good to have: Understanding of document stores (e.g.,
  MongoDB)
