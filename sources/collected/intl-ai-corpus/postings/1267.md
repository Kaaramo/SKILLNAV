Job Description – Junior Full Stack Engineer (AI Specialist)
Location: Kochi / Onsite

Experience: 4–8 years

Role Overview

We are looking for a motivated Full Stack Engineer with AI
API integration skills to support mission-critical
applications. The role requires flexibility to work in 24x7
rotational shifts, proficiency in Python, and readiness to
troubleshoot, enhance, and fix bugs while providing L2-level
support.

Key Responsibilities

- Develop, test, and deploy full stack applications with AI/ML
  features.
- Implement and support AI API integrations across Claude,
  OpenAI, Gemini (Google), and Azure AI services.
- Provide L2-level support: monitor applications, maintain
  system health, and resolve issues quickly.
- Focus on bug fixing, troubleshooting, and incremental
  enhancements to existing applications.
- Collaborate with global teams using GitHub workflows for
  version control and CI/CD.
- Participate in agile ceremonies, code reviews, and
  documentation.
- Continuously learn and adopt new AI engineering concepts and
  cloud technologies.
- Knowledge about Incident Management, Problem Management,
  Service requests and change requests.

Required Skills & Expertise

- Full Stack Development: React/Angular, Node.js, Python.
- AI Engineering: Knowledge of AI API implementation and
  integration patterns.
- AI APIs: Exposure to Claude, OpenAI, Gemini, Azure.
- Cloud & Data: Azure (preferred), REST APIs, relational/non-
  relational databases.
- Collaboration Tools: GitHub for code management and CI/CD.
- Support Knowledge: Familiarity with production support
  practices and ITIL processes is an added advantage.
- Shift Readiness: Willingness to work in rotational shifts
  for 24x7 support.
- Soft Skills: Strong communication, problem-solving, and
  teamwork abilities.
