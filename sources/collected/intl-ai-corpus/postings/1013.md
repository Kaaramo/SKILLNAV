Job Purpose

The Senior Power Platform Engineer will spearhead the
strategic implementation and integration of Microsoft
Copilot across our enterprise applications and workflows.
This key role will drive the adoption of AI-powered
assistance to significantly enhance user productivity and
streamline business processes.   This will combine
innovative chat bot solution with content from SharePoint
document libraries or structured data from the MS Power
Platform and leverage the newest agent capabilities to turn
GenAI content into value added actions.

Key Accountabilities:

Develop AI Solutions: Implement AI solutions using Azure AI,
Co Pilot Studio, and other Microsoft Azure AI services to
solve complex business problems.

Azure AI Services: Utilize a range of Azure AI services
(e.g., Azure Cognitive Services, Azure Machine Learning,
Azure Bot Services) to develop and deploy intelligent
applications.


Develop Low Code Solutions: Utilize the Microsoft Power
Platform (Model driven Power Apps, Power Automate, Power BI,
Power Pages) and MS SharePoint. applications to create low
code solutions that integrate seamlessly with existing
business processes and workflows.
