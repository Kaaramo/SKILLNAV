ABOUT US

Xsolla is a global commerce company with robust tools and
services to help developers solve the inherent challenges of
the video game industry. From indie to AAA, companies
partner with Xsolla to help them fund, distribute, market,
and monetize their games. Grounded in the belief in the

game.

Title: Full Stack AI Engineer

What You’ll do:

- Design and develop full-stack applications that power GenAI
  applications and platforms

- Develop and maintain frontend applications using Next.js
  with API routes.

- Design and implement visually appealing and responsive user
  interfaces using Tailwind CSS / Shadcn or other CSS
  libraries.

- Write clean, efficient, and scalable code in Python and
  TypeScript.

- Work with various libraries or frameworks based on project
  requirements.

- Be up-to-date with latest happenings in AI and Associated AI
  tools, technologies and frameworks (e.g. MCP, Agentic
  systems)

- Build APIs and infrastructure to support LLM-based features,
  tools, and agent workflows.

- Collaborate with frontend, ML, and product teams to create
  seamless AI experiences.

- Optimize backend systems for performance, scalability, and
  reliability.

- Work with vector databases, data pipelines, and AI model
  endpoints (OpenAI, Anthropic, Hugging Face, etc.).

- Maintain clean, secure, and production-grade code

What We’re Looking For:

- 1–3 years of experience in full-stack development.

- Proficient in Python and Typescript.

- Strong experience with REST APIs, microservices, and

- Experience building or supporting GenAI applications and/or
  LLM-powered products.

- Exposure to cloud services (AWS, GCP, or Azure) and
  containerization (Docker, Kubernetes)

- Excellent problem-solving, communication, and collaboration
