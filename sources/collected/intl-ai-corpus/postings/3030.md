### Raisonnement classification (LLM upstream)

This role is primarily a full stack software engineering
position that integrates pre-built AI/ML models and LLM APIs
into web applications. The developer builds UIs and backend
services, embedding AI capabilities built by others (data
scientists, ML engineers) into end-user apps. While it
involves working with AI/LLM services, the core work is
application development, not building, training, or fine-
tuning models directly.
