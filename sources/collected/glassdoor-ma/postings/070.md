**TradeAI Agency** is building an AI-powered SaaS platform focused on ICT/SMC market analysis and smart trading tools for traders worldwide.

We are looking for a passionate AI/Python Developer to join our remote startup project and help us build innovative trading analysis systems using AI and automation.

This is a remote opportunity where we first want to organize an online meeting to get to know each other, discuss the project vision, and explore long-term collaboration.

_**Responsibilities**_

- Develop backend systems using Python
- Build AI-powered market analysis tools
- Integrate chart and market data APIs
- Work on ICT/SMC detection logic
- Build automation systems and APIs
- Collaborate on AI chart analysis features
- Optimize performance and scalability

_**Requirements**_

- Strong experience with Python
- Experience with FastAPI or Django
- Knowledge of AI APIs and automation
- Experience working with APIs
- Understanding of trading/financial markets is a plus
- Problem-solving mindset
- Good communication skills
- Ability to work remotely

_**Nice To Have**_

- Experience with TradingView APIs
- Knowledge of machine learning
- Computer vision experience
- Interest in trading or fintech

_**Job Type**_

- Remote
- Freelance / Contract
- Potential long-term collaboration

_**Hiring Process**_

- First online meeting
- Technical discussion
- Small test/project discussion
- Collaboration start
